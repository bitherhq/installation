Install on Gnu/Linux

Installing

  1. Download the latest version of BITH.
  2. Copy it in your bin directory
  3. Run it!

You should get something like this:

$ wget LINK
$ sudo mv bith /usr/bin
$ bith

Installing on ubuntu by PPA

  1. Add bither PPA and then update your repositories.
  2. Install bith
  3. Run it!
$ sudo add-apt-repository ppa:bither/bither
$ sudo apt update
$ sudo apt install bith
